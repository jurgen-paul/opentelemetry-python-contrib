OpenTelemetry pulsar Instrumentation
=============================================

|pypi|

.. |pypi| image:: https://badge.fury.io/py/opentelemetry-instrumentation-pulsar.svg
   :target: https://pypi.org/project/opentelemetry-instrumentation-pulsar/

This library allows tracing requests made by the pulsar library.

Installation
------------

::

    pip install opentelemetry-instrumentation-pulsar
